# Director Blocking

Use this reference for shot grouping, director staging, camera, transitions, time feasibility, and asset reverse-planning.

## Blocking Preview Fields

Before writing a video prompt, preview:

- script segment and original dialogue
- shot purpose
- time range and duration
- character positions
- scene geography
- foreground / midground / background
- shot size
- lens or camera feel
- camera movement
- action beat
- micro-expression
- hand and prop action
- environmental motion
- light behavior
- transition in
- transition out
- end hold
- sound bridge
- risk points

## Time Feasibility

For every 15-second segment, check:

- number of shots
- number of actions
- number of dialogue lines
- whether the line can be spoken naturally
- whether camera movement and action compete
- whether the end hold has at least 0.5-1 second

If too dense, split the shot group or reduce actions.

## Dialogue Must Be Checked

If the project comes from a script, always check original dialogue during:

- script handoff
- shot grouping
- director blocking
- time feasibility
- video prompt writing

Do not drop dialogue silently. If a shot has no dialogue, say why silence is intentional.

## Director Prompt Style

Write like a director, not like a plot summary:

- what the camera sees now
- what changes in frame
- where attention moves
- what stays still
- how the action ends
- what sound leads the cut

Avoid explaining lore unless it changes visible action.

## Transition Rules

For every shot or segment, specify:

- cut, dissolve, match cut, action cut, sound bridge, focus shift, occlusion wipe, or hold-to-cut
- why this transition is used
- what visual or sound element connects to the next shot

If uncertain, use stable cuts and sound bridges.

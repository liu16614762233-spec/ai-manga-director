# Sound And Dialogue

Use this reference for dialogue design, narration, voice consistency, Foley, ambience, and final music planning.

## Dialogue Director Rule

Dialogue must come from the script or visible dramatic need. Do not add lines because a shot feels empty.

For every segment check:

- original script line
- speaker
- line function
- time available
- mouth visibility
- voice consistency
- whether silence is stronger

Line functions:

- reveal information
- command action
- create threat
- expose emotion
- bridge scenes
- deliver hook

## Narration Rule

Narration is optional. Use it only when:

- the image cannot carry necessary context
- the story needs a time jump
- a final thematic line is required
- the target format expects narration

Do not use narration to explain what is already visible.

## Raw Generation Sound

For video generation prompts:

- include dialogue if needed
- include environment sound
- include Foley
- include object sound
- include sound bridge
- forbid background music

## Final Edit Music

Background music can be planned for final editing, but keep it out of raw model generation unless the user explicitly wants generated music in the clip.

Separate:

- raw clip sound
- final edit score

## Voice Consistency

For each recurring speaking character record:

- voice age
- pitch
- tempo
- texture
- emotional baseline
- distance / room feel
- accent or language style

If unavailable, write a temporary voice anchor and mark it unapproved.

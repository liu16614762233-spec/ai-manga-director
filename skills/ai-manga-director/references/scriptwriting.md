# Scriptwriting

Use this reference when the user wants to write or develop a story before production.

## Writing Room Boundary

Writing can run without production assets. Keep it separate from visual production until the user asks to hand off.

Writing outputs may include:

- premise
- logline
- theme
- world rules
- character arcs
- episode outline
- scene outline
- dialogue pass
- narration pass
- pacing pass

Do not force asset gates during pure writing.

## Production Handoff

When the user asks to move from writing to production, output:

- confirmed project facts
- unconfirmed assumptions
- character list
- necessary scenes
- key props / creatures / vehicles
- original dialogue by segment
- visual style requirement
- sound requirement
- first asset gate list

Then route to workflow gates.

## Dialogue Pass

Check:

- who speaks
- why they speak now
- what changes after the line
- whether the image already says it
- whether the line fits available time

Prefer fewer lines with stronger function.

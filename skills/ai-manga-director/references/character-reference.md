# Character Reference

Use this reference for character reference images, look development, and character asset gates.

## Default Character Board

When the user asks for:

- character reference
- character design sheet
- character look
- character setup
- default character still
- main character design

and does not explicitly request another format, output the professional character reference board:

```text
9:16 vertical professional character reference board.
Strict unequal 2x2 grid.
Horizontal split at exactly 20% from the top.
Top row is 20% height: narrow front face close-up and narrow side face close-up.
Bottom row is 80% height: front body from collarbone down with no head or face, and full back view from head to feet.
The lower row is the main visual area.
```

Lock:

- same character identity
- same costume
- same materials
- same skin tone
- same hair
- same headpiece
- same height impression
- same head-body ratio
- same shoulder width
- same waistline
- same leg length
- same arm length
- same body thickness
- same footwear baseline
- same accessory position

Forbid:

- equal four-grid
- ordinary three-view
- multi-pose collage
- action pose
- poster composition
- text, labels, logo, watermark
- visible face/head in the lower-left body panel

## Supplemental Character Assets

Use these only when needed:

- three-view: front / side / back for full-body structure
- proportion ruler: height and body ratio consistency
- costume breakdown: layers, materials, accessories
- expression sheet: facial performance
- micro-expression board: subtle emotional transitions
- action pose sheet: body motion
- hand / prop detail: contact and grip

These never replace the default character board unless the user explicitly requests them.

## Review Gate

Reject or mark review if:

- the board is equal 2x2
- top row exceeds 20%
- lower-left panel includes a face, chin, mouth, nose, eyes, or head
- lower body is too small
- front and back proportions differ
- costume or headpiece changes between panels
- it looks like a poster, action sheet, or normal three-view

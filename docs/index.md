# AI Manga Director

AI Manga Director is a universal agent workflow pack for creators who want a repeatable AI manga and cinematic video production pipeline.

It focuses on the step that most prompt workflows skip: director-level production control.

## Core Promise

Turn this:

```text
Here is my script. Make video prompts.
```

into this:

```text
Project facts checked.
Main characters and scenes gated.
Director blocking preview complete.
Minimum control images selected.
Seedance-style video prompt ready.
Review and rework path prepared.
```

## Main Modules

- Writing room
- Project workflow gates
- Director blocking
- Character reference boards
- Asset and control-image planning
- Seedance-style video prompts
- Image refinement
- Dialogue and sound
- Review and rework

## Design Position

This skill is for production, not inspiration browsing. It is built to reduce:

- identity drift
- scene drift
- overlong prompts
- missing asset gates
- ignored dialogue
- fake-looking keyframes
- uncontrolled effects
- full redesign after minor sample failures

## Recommended First Prompt

```text
Use AI Manga Director. Run compact mode. First identify my task type, current gate, missing required assets, and the next allowed action.
```

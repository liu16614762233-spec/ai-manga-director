Why It Exists

Not a prompt dump

It prevents the common jump from plot directly to a long model prompt.

Built for continuity

Character references, scene masters, props, dialogue, and gates are checked before generation.

Model-aware

Reference responsibilities, control images, raw sound limits, and rework scope are explicit.

Production Workflow

Route the task and check visible project facts.
Run gates for characters, scenes, props, dialogue, and timing.
Build director blocking before writing model prompts.
Plan only the minimum necessary production assets and control images.
Write compact Seedance-style prompts with reference responsibilities.
Review generated images and video samples.
Rework while preserving useful parts of the existing sample.
Propose continuity updates only after user approval.

Core Modules

Writing Room

Story, outline, scene, dialogue, narration, and production handoff.

Director Blocking

Camera, staging, action, micro-expression, transition, end hold, and sound bridge.

Control Images

Storyboards, spatial maps, action paths, camera guides, effect zones, and start/end frames.

Character Reference

Default 9:16 unequal 2x2 professional character board with strict proportion lock.

Image Refinement

M0-M5 routing from no-op to cinematic realism and model-reference readiness.

Review And Rework

Diagnose failures and change only the highest-risk item first.

Universal Startup Prompt

Works with Custom GPTs, Claude-style Skills, Codex, Gemini/Grok-style agents, and local agent frameworks that can read Markdown instructions.

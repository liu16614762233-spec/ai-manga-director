# Workflow Preview

## 1. Start

Input can be a script, scene, project package, image, generated video, or draft prompt.

## 2. Route

The skill identifies whether the task belongs to writing, project setup, asset planning, director blocking, video prompt writing, image refinement, sound, or review.

## 3. Gate

The skill checks whether the requested next step is allowed.

Hard blockers include missing main-character references, missing required scene masters, missing prop/monster/vehicle designs, missing original dialogue check, unclear reference responsibilities, or failed time feasibility.

## 4. Block

The skill builds the shot as a director would:

- subject position
- camera position
- lens / shot size
- foreground and background
- action beat
- micro-expression
- hand and prop action
- environment motion
- sound bridge
- transition
- ending state

## 5. Plan Images

The skill chooses the minimum necessary images:

- design assets for appearance
- storyboards for sequence
- spatial maps for object relationships
- action paths for movement
- camera guides for complex motion
- effect-zone maps for smoke, fire, fog, water, dust, or spread
- start/end frames only when exact states matter

## 6. Prompt

The skill writes compact model-facing prompts with reference responsibilities, timing, sound, color, ending, and forbidden items.

## 7. Review

The skill reviews images or videos and recommends the smallest useful fix, preserving approved samples whenever possible.
